# Code of Conduct

This project adopts the **Contributor Covenant, v2.1** as its code of conduct.

The full text is maintained upstream and referenced here by link rather than
duplicated in-tree so updates land automatically:

- **Full text:** https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- **Machine-readable:** https://www.contributor-covenant.org/version/2/1/code_of_conduct.md
- **FAQ:** https://www.contributor-covenant.org/faq/
- **Translations:** https://www.contributor-covenant.org/translations/

## Scope

The Covenant applies to all project spaces — the GitHub repository, issues,
pull requests, discussions, code comments, and any off-platform venue where
someone represents the project.

## Reporting

Conduct concerns go to the maintainer via the same private channel used for
security reports — see [SECURITY.md](./SECURITY.md) for the preferred
GitHub-private-advisory path and the email fallback.

Reports are treated as confidential. The maintainer will acknowledge within
72 hours and respond substantively within 5 business days.

## Enforcement

Consequences follow the Contributor Covenant **Community Impact Guidelines**
(section 5 of the linked text) — correction, warning, temporary ban, or
permanent ban, scaled to the severity and pattern of the behaviour.
